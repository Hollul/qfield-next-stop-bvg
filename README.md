Just a little test plugin for [QField](https://qfield.org/). It shows the most nearby station of [BVG](https://www.bvg.de) (Berlin public transport) in Berlin, Germany and the distance to it.

This plugin uses the [BVG Rest API](https://v6.bvg.transport.rest/api.html) and icons from [OpenMoji](https://openmoji.org/).

## Installation

To install the plugin, just copy the download link to the zip file of this repo and paste it in QField Plugin installation.
